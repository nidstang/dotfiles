# Cuándo usar esta skill
Usála cuando el usuario quiera empezar una nueva tarea y te de un código de tarea

# Instrucciones
- Usa el MCP `jira` para localizar el ticket, comentarios y contexto.
- No uses otros MCP salvo que el usuario lo pida explícitamente.
- Saca todo el contexto de la tarea con el código a tocar en el repositorio donde estemos
- Plantea un plan inicial de que es lo que se pide hacer
- Si falta contexto, consulta de nuevo el MCP `jira`.
